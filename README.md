
# Dynamic DashBoard




### Author - [@Sara-Kulsum](https://github.com/SaraKulsum)

 


### Run Locally

#### Clone the project

```bash
  git clone https://github.com/SaraKulsum/Dynamic-Dashboard-SaraKulsum.git
```

#### Go to the project directory

```bash
  cd Dynamic-Dashboard-SaraKulsum
```

#### Install dependencies

```bash
  npm install
```

#### Start the server

```bash
  npm run dev
```
